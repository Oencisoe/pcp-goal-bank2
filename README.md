# PCP Goal Bank

An interactive Person-Centered Plan goal bank with 361 pre-written outcomes across 15 domains, mapped to 23 assessment frameworks including CDER, OASIS, FIM, Barthel, GMFCS, MACS, CFCS, EDACS, ABAS-3, Vineland-3, SIS, QOLIE, and more.

## Features
- Search and filter goals by keyword or assessment framework
- Click to copy individual goals
- Select All for bulk export
- Export selected goals to Word document
- Built-in Assessment Reference Library with 40+ links to PDFs and tools

## Setup
```bash
npm install
npm run dev
```

## Deploy to GitHub Pages
Push to `main` branch — GitHub Actions will auto-deploy.

Your site will be at: `https://YOUR-USERNAME.github.io/pcp-goal-bank/`
